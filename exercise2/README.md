# Quan ly nguoi dung

Project React quan ly nguoi dung voi day du chuc nang CRUD.

## Cai dat

```bash
npm install
```

## Chay project

```bash
npm start
```

Project se chay tai [http://localhost:3000](http://localhost:3000)

## Cau truc project

- `src/App.js` - Component goc App
- `src/components/SearchForm.js` - Component tim kiem
- `src/components/AddUser.js` - Component them nguoi dung
- `src/components/ResultTable.js` - Component hien thi danh sach, sua, xoa
